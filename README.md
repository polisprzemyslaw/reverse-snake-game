# Reverse Snake Game

A unique twist on the classic Snake game where the goal is to survive as long as you can while being hunted by snakes.

## Live Demo
**[Click here to play the game in your browser!](https://polisprzemyslaw.github.io/reverse-snake-game/)**

## How to Run Locally
1. Download the `index.html` file.
2. Open it in any modern web browser (Chrome, Firefox, Edge).

## Gameplay
In this version, **you are the food**. 
* **Your Role:** You play as a red piece of food.
* **Controls:** Move using the **Arrow Keys**.
* **Goal:** Avoid the snakes for as long as possible.
* **Mechanics:** Snakes spawn randomly and their difficulty increases over time. Black pieces of food spawn occasionally - you can use them as a distraction to lead snakes away from you.
